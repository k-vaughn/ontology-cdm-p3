# Travel Corridor Pattern

The travel corridor pattern specializes the transport network pattern to address the needs for a more holistic travel view.

It consists of the following classes:

- [Travel Corridor](transportnetwork__TravelCorridor.md)
- [Travel Corridor Link](transportnetwork__TravelCorridorLink.md)
- [Travel Corridor Segment](transportnetwork__TravelCorridorSegment.md)
