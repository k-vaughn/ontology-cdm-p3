# Micromobility Network Pattern

The micromobility network pattern specializes the road network pattern to address the needs for the travel of micromobility vehicles.

It consists of the following classes:

- [Micromobility Lane](transportnetwork__MicromobilityLane.md)
- [Micromobility Link](transportnetwork__MicromobilityLink.md)
- [Micromobility Network](transportnetwork__MicromobilityNetwork.md)
- [Micromobility Path](transportnetwork__MicromobilityPath.md)
- [Micromobility Path Section](transportnetwork__MicromobilityPathSection.md)
- [Micromobility Path Segment](transportnetwork__MicromobilityPathSegment.md)
