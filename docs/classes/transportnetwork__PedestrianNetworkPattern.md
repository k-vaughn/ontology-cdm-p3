# Pedestrian Network Pattern

The pedestrian network pattern specializes the transport network pattern to address the needs for pedestrian travel.

It consists of the following classes:

- [Footpath](transportnetwork__Footpath.md)
- [Footpath Lane](transportnetwork__FootpathLane.md)
- [Footpath Link](transportnetwork__FootpathLink.md)
- [Footpath Network](transportnetwork__FootpathNetwork.md)
- [Footpath Section](transportnetwork__FootpathSection.md)
- [Footpath Segment](transportnetwork__FootpathSegment.md)
